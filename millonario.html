1. Fundamentos Lógicos

¿Qué es un Algoritmo?

Es una secuencia de pasos lógicos, finitos y ordenados para resolver un problema. En esta clase usamos Algoritmos Secuenciales, los cuales se ejecutan instrucción por instrucción, de arriba hacia abajo, sin saltarse nada.

  • Paso 1: Entrada (Pedir y leer datos)
  • Paso 2: Proceso (Calcular con operaciones matemáticas)
  • Paso 3: Salida (Mostrar el resultado en pantalla)

Las Variables

Son espacios en la memoria de la computadora donde guardamos datos que pueden cambiar. Imagínalas como cajas con una etiqueta. Reglas clave para nombrarlas:

  • Deben empezar con una letra (ej. area, base_salon).
  • No pueden tener espacios en blanco (mal: mi area).
  • No usar comandos reservados (mal: Escribir, Leer).

2. Datos y Matemáticas

Entero

Números exactos sin parte decimal.

Ej: 10, -5, 2024
Real

Números con punto decimal para mayor precisión.

Ej: 3.14, 15.5, -0.5
Cadena

Texto o palabras. (Siempre van entre comillas).

Ej: "Hola", "Juan"
Operadores

Signos matemáticos en PSeInt:

+ (Suma), - (Resta)
* (Multiplicación)
/ (División)

3. PSeInt y Diagramas de Flujo

Definir

Crea la variable en memoria y le asigna un tipo de dato.
Ej: Definir base Como Real

Escribir

Muestra un mensaje o el valor de una variable en pantalla.
Ej: Escribir "Ingrese altura:"

Leer

Pausa el programa y espera a que el usuario escriba un dato en el teclado para guardarlo.
Ej: Leer altura

<- (Asignación)

Guarda el resultado de un cálculo matemático directo en una variable.
Ej: area <- base * altura

Figuras del Diagrama de Flujo

  • Inicio/Fin
    Óvalo: Marca el Inicio y el Fin absoluto del algoritmo.
  • Datos
    Paralelogramo: Representa Entrada (Leer) y Salida (Escribir).
  • Proceso
    Rectángulo: Representa un Proceso Interno (Cálculos matemáticos).
  • Líneas de flujo: Conectan los pasos indicando la secuencia obligatoria de ejecución.

4. Ejemplos Prácticos Completos

Ejemplo: Suma de Números 

Proceso SumaNumeros
    // 1. Definir variables en memoria
    Definir A, B, C Como Real
    
    // 2. ENTRADA de datos
    Escribir "Ingrese el primer número:"
    Leer A
    Escribir "Ingrese el segundo número:"
    Leer B
    
    // 3. PROCESO matemático
    C <- A + B
    
    // 4. SALIDA del resultado
    Escribir "El resultado de la suma es: ", C
FinProceso

Ejemplo: Área del Salón 

Proceso AreaRectangulo
    // 1. Definir variables (base, altura y area)
    Definir base, altura, area Como Real
    
    // 2. ENTRADA de medidas
    Escribir "Ingrese la base del terreno:"
    Leer base
    Escribir "Ingrese la altura del terreno:"
    Leer altura
    
    // 3. PROCESO (Multiplicación)
    area <- base * altura
    
    // 4. SALIDA del resultado final
    Escribir "El área del salón es: ", area
FinProceso
